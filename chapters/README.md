## Cheat sheet summary
- `01-introduction.md`: Quick GitHub Actions context and workflow syntax overview
- `02-certification-objectives.md`: Certification-oriented cheat sheet for triggers, contexts, security, and reuse
- `03-learning-path.md`: Detailed contexts and expressions cheat sheet with examples
- `04-study-plan.md`: Study-focused patterns for event filters, matrix builds, outputs, and conditionals
- `05-hands-on-exercises.md`: Practical workflow snippets for push CI, matrix jobs, artifacts, and reusable workflows
- `06-project-ideas.md`: Project planning cheat sheet for pipelines, security, caching, and observability
- `07-exam-topics.md`: Exam topic cheat sheet with trigger, context, conditional, and security guidance
- `08-sample-workflows.md`: Sample workflow cheat sheet for CI, reusable workflows, composite actions, self-hosted runners, and deployment
- `09-resources.md`: Resource cheat sheet for docs, labs, example repos, and best-practice references
- `10-mock-preparation.md`: Mock exam cheat sheet with practice tasks for triggers, outputs, secrets, reuse, and debugging
- `11-production-focus.md`: Production readiness cheat sheet with workflow hygiene, security, reliability, observability, and maintenance
